## Environment setup (from scratch)

**1. Install uv** (skip if already installed):
```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

**2. From the repo root** (`curation/`), create the environment and install all dependencies:
```bash
uv python install 3.9      # download Python 3.9 if not present
uv venv --python 3.9       # create .venv/ in the repo root
uv pip install numpy pandas scikit-learn matplotlib scipy \
               ipykernel jupyter notebook \
               gensim fastparquet
```

`.venv/` is created in the repo root and picked up automatically by all `uv run` commands.

## Running notebooks

**Option A — VSCode (recommended)**

1. Install the [Jupyter extension](https://marketplace.visualstudio.com/items?itemName=ms-toolsai.jupyter) (`ms-toolsai.jupyter`) in VSCode.
2. Open any `.ipynb` file in `code/` directly in VSCode.
3. Click **Select Kernel** (top-right) → **Python Environments** → choose `.venv` from the repo root.
4. Run cells with `Shift+Enter` or **Run All**.

**Option B — browser**

From the repo root (`curation/`):
```bash
uv run jupyter notebook code/
```
Then open any notebook and run all cells (`Kernel → Restart & Run All`).

No data download needed — all notebooks load from the pre-sampled 25K file already in the repo.

## Spotify dataset

- All notebooks use a Kaggle dataset that can be manually downloaded from [kaggle.com/datasets/bwandowando/spotify-songs-with-attributes-and-lyrics](https://www.kaggle.com/datasets/bwandowando/spotify-songs-with-attributes-and-lyrics). 

- For the purpose of the class, no download needed to run the notebooks. We load a pre-sampled 25K file at `data/spotify/sample_songs.csv`, which is regenerated by the commented-out cells in `0.data_exploration.ipynb`.  
- In case you want to check the full dataset (~955K tracks, ~2 GB): 
   1. Go to the dataset page: [kaggle.com/datasets/bwandowando/spotify-songs-with-attributes-and-lyrics](https://www.kaggle.com/datasets/bwandowando/spotify-songs-with-attributes-and-lyrics).
   2. Click **Download** (requires a free Kaggle account).
   3. Unzip the downloaded archive and place `songs_with_attributes_and_lyrics.csv` at:
   ```
   data/spotify/songs_with_attributes_and_lyrics.csv
   ```
